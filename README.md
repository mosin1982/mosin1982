## Architecture Principles

```text
NO DATA  →  NO DECISION
CONFLICT →  HOLD / VERIFY
EVIDENCE →  TRACEABLE OUTPUT
SYSTEMS  →  MODULAR BY DESIGN
```

I value:

- Evidence before conclusions
- Fail-safe behavior over confident guessing
- Modular systems over rigid monoliths
- Observability, traceability, and validation
- Privacy-aware and responsible engineering
